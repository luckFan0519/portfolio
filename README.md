# 姜凡个人主页

云南大学智能科学与技术本科生个人作品集页面。

## 在线访问

**https://luckfan0519.github.io/portfolio/**

## 项目结构

```
portfolio/
├── index.html              # 主页面
├── assets/
│   ├── css/style.css       # 样式文件
│   ├── js/main.js          # 交互脚本
│   └── images/             # 图片资源
│       ├── avatar.png
│       ├── yunnan-university.png
│       ├── project-xiaotong.png
│       ├── project-ai.png
│       └── project-signlanguage.jpg
└── README.md
```

## 技术栈

- HTML5 / CSS3 / Vanilla JavaScript
- Canvas 粒子背景
- IntersectionObserver 滚动动画
- CSS Glassmorphism 玻璃拟态
- 3D Tilt 卡片效果
